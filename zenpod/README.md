# ZenPod Landing Page

A premium, Apple-style landing page for a relaxation and focus wearable.

## Key Features
- **Glassmorphism:** Elegant use of blur and transparency.
- **Product Showcase:** High-resolution image placement with clean typography.
- **Interactive FAQ:** Accordion-style FAQ section.
- **Visual Hierarchy:** Large, breathy margins to reflect the "Zen" brand identity.

## Tech Stack
- HTML5
- Vanilla CSS
- Vanilla JS
