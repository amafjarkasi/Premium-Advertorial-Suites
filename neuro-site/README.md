# Neuro Landing Page

A high-tech, science-focused landing page for a nootropic supplement.

## Key Features
- **Modern UI:** Dark mode aesthetic with blue and cyan accents.
- **Scientific Storytelling:** Sections dedicated to the "Brain-Computer Interface" and "Cellular Rebirth".
- **Interactive Stats:** Data-driven presentation of clinical results.
- **Micro-Copy:** Conversion-optimized headlines and subheadlines.

## Tech Stack
- HTML5
- Vanilla CSS (Custom Grid/Flexbox)
- Vanilla JS
