# GlowFactor Landing Page

A high-converting, direct-response landing page for a premium skincare brand.

## Key Features
- **Advertorial Layout:** Designed to look like a health/science article to build trust.
- **Micro-Animations:** Subtle entrance effects and hover states.
- **Conversion Tools:** Sticky CTA bar, live viewer count, and scrolling "stock" scarcity notes.
- **Mobile Responsive:** Optimized for mobile browsing.

## Tech Stack
- HTML5
- Vanilla CSS
- Vanilla JS
